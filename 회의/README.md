### 기능
- 일단은 패스 안붙히고 추후 필요 시 개발 예정(현재 플라스틱 수치 사용자에게 전송 예정이라 저장할 필요 못느낌)

### 역할
- 설계도 : 정지민
- OpenCV : 장동건
- 카메라 측정 환경 설정 (카메라 조절) : 유준선
- 텔레그램 : 류경록
- 서류 종합 및 서포트 : 박종호

### 예상 과정
일정 시간 별로 측정 -> 학습된거 이용해서 OpenCV 분석 및 출력 값 변환 -> 텔레그램 전송

### 고려 사항
- 비율 어떻게 출력할건지??
현재 생각  
- 나오는 개수 : 큰거 하나와 작은거 하나의 차이점 구별 힘듬
- 화면에 찍힌 비율 : 수치상으로 사용자가 이해하기 어려울 수 있음
- 개수,비율 계산해서 특정치 치환 : 일단 CV결과 봐야 암. 현재 이거 할 계획
